# AMUSIC

### 概要
###### 誰でも音楽を学べて音楽に触れられるアプリ
プロの演奏でも知り合いの演奏でも、演奏会などに行く機会は多いと思います。
酷い演奏だという感想を持つ人はいませんよね？
多くの人は「良い演奏だった」、「機会があればまた聞きに来たい」など前向きな感想を持つと思います。
中には「楽しそう」、「やってみたい」と思う方もいますよね？そんな方のために作りました。
<br>もう歳だから、、全く知識がないから、、
<br>そんなことはありません！音楽はいつから始めても良いんです。

少しでも興味を持っている方、その興味を引き上げるために使って欲しくて作りました。

## 利用方法
クイズで学び、チャットで質問し、ゲームで遊ぶことができる。
<br>チャット以外はアカウント登録をしなくても体験できる。

## 目指した課題解決
「音楽に興味はあるけど今更始めるのは気が引ける」、「学びたいけど面倒臭い」という人のために少しでも音楽を楽しんでもらえるように作った。ここから初めてさらに興味を持って欲しい。

## 実装した機能
ユーザー機能：アカウント登録をするとチャット機能を使うことができる。
<br>マイページ：ユーザーの情報を見ることができ、そのユーザーが投稿したチャットやお気に入り登録した投稿を見ることができる。
<br>チャット機能：チャットを投稿できる。一覧画面ではチャットの一覧を見ることができ、詳細画面ではコメントやお気に入り登録ができる。チャット投稿者は編集や削除ができる。
<br>クイズ機能：すでに作ってある問題にランダムで、1度に10問の問題に挑戦できる。最後には成績が開示され、もう1度挑戦するかトップページに戻るかを選択できる。
<br>コメント機能：チャットの詳細画面でコメントできる。コメント時にコメントした時間とユーザーも表示される。
<br>質問検索機能：検索するとチャットの中から条件を満たすチャットのみが表示される。
<br>音がなるボタン：少しでも楽しめるように実装。任意のボタンを押した時ランダムで音が鳴る。音を鳴らすときは音がデカすぎないよう注意する必要あり。
<br>お気に入り機能：気に入ったチャットがあればチャットの詳細画面でお気に入り登録できる。もう1度押すとお気に入り解除できる。
<br>タグ付け機能：chat投稿の際タグをつけて投稿すると、chat検索の際タグで検索をするとそのカテゴリーに合うchatのみが表示される

## 実装予定の機能
ユーザー編集機能：ユーザー情報の名前とメールアドレスを編集することができる。
<br>ゲーム機能：リズムゲームや音当てゲームを実装したいが、今は実装できる力がないので勉強中。
<br>コメント機能：非同期通信でコメントできる。
<br>質問検索機能：タグごとに検索できる。
<br>クイズ作成機能：ユーザーは自分でクイズを作り挑戦することができる。作った問題に挑戦するか全体の問題に挑戦するか選ぶことができる。
